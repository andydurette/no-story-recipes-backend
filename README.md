## Usage

Repo of code that can acts as the backend for recipe related requests and holds the recipes in db to be called.

This repo utilizes [Nest](https://github.com/nestjs/nest) framework

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
